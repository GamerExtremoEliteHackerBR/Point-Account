A aplicação permite o registro e gerenciamento de jornadas de trabalho de forma intuitiva e eficiente, funcionando 100% no navegador sem necessidade de servidor ou internet (após o primeiro carregamento).

Solução moderna e acessível para controle de ponto, ideal para:

## 🎯 Sobre o Projeto

O **Conta Ponto** é uma solução gratuita e profissional para gerenciamento de jornada de trabalho. Desenvolvido com tecnologias web padrão (HTML5, CSS3, JavaScript puro), o sistema oferece uma experiência completa de controle de ponto sem necessidade de servidor, banco de dados ou conexão com internet.

### ✨ Destaques

- ✅ **Sem dependências externas** - Vanilla JS puro
- ✅ **Offline First** - Funciona sem internet
- ✅ **PWA** - Instalável como aplicativo nativo
- ✅ **Design Moderno** - Interface inspirada no iOS
- ✅ **Temas** - Claro e Escuro
- ✅ **Backup** - Exportação e importação de dados
- ✅ **Privacidade** - Dados salvos apenas localmente

## 🚀 Funcionalidades

### 📋 Registro de Ponto
- Registrar entrada/saída com horas, minutos e segundos
- Edição de horários existentes
- Comentários por registro
- Cálculo automático de total e saldo
- Saída prevista baseada na jornada
- Dias de folga programados

### 📊 Relatórios
- Filtro por semana, mês ou intervalo personalizado
- Totais consolidados do período
- Lista detalhada por dia
- Exportação para CSV (Excel/Google Sheets)
- Navegação entre períodos

### ⚙️ Configurações
- **7 jornadas pré-definidas**:
  - Segunda a Sexta (08:48)
  - Segunda a Sábado (08:00)
  - Domingo a Sexta (07:30)
  - Plantão 12x36 (12:00)
  - Semana Inglesa (07:20)
  - Jornada Reduzida (06:00)
  - Jornada Estendida (09:00)
- Jornada personalizável
- Tema claro/escuro
- Notificações
- Backup/Restauração
- Limpeza total de dados

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Design system com CSS Variables
- **JavaScript (ES6+)** - Lógica e interatividade
- **LocalStorage API** - Persistência de dados
- **PWA** - Manifest.json para instalação
